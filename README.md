# Benzine usage api
This api handles requests for the benzine usage app.
Its a REST api that uses mongoose for the connection with mongodb.

## Installation

Install the node modules
```bash
npm install
```

## Usage

Start nodejs api
```bash
node app
```

Generate swagger documentation
```bash
npm run docu
```
